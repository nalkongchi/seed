# 🌾 나는야 종자검사원
> 검사원이 되기 위한 용사의 모험 — 국립종자원 교육 RPG 게임

---

## 📖 게임 소개

종자 왕국의 질서를 되찾기 위해 떠나는 픽셀 아트 RPG 교육 게임입니다.
플레이어는 종자검사원 지망생으로서 6개의 스테이지를 돌파하며
올바른 합격/불합격 판정 능력을 기릅니다.

---

## 🎮 게임 흐름

```
타이틀 → 이름 입력 → 오프닝 → 지도 → 전투 → 클리어/게임오버 → 엔딩
```

- 지도에서 스테이지를 선택하면 전투 진입
- 적이 종자 관련 주장을 하면 **합격 / 불합격** 판정
- 틀리면 체력 감소, 5문제 모두 클리어하면 다음 스테이지 해금
- 최종 보스(심판의 성)를 격파하면 정식 종자검사원 임명

## ✏️ 문제 추가/수정 방법

`data.js` 파일만 수정하면 됩니다. 코드 지식 없이도 가능합니다.

### 문제 1개의 구조

```javascript
{
  text: '"문제 내용을 여기에 씁니다."',
  answer: 'pass',   // 합격이면 'pass', 불합격이면 'fail'
  reason: '판정 근거 설명을 여기에 씁니다.'
}
```

### 문제 추가 규칙

- 각 스테이지의 `questions` 배열에 객체를 추가합니다
- 게임은 매 전투마다 전체 문제 중 5개를 **랜덤으로** 뽑아 출제합니다
- 스테이지당 최소 5개 이상, **50개까지 추가를 권장합니다**

### 줄바꿈 자동 처리

문제 텍스트에서 마침표(`.`) 물음표(`?`) 느낌표(`!`) 뒤 띄어쓰기가 있으면
게임이 자동으로 줄바꿈 처리합니다. 별도 수정 불필요합니다.

## 🖼️ 적 이미지 교체 방법

1. `images/` 폴더에 이미지 파일 넣기 (PNG 권장, 정사각형)
2. 경로는 이미 `data.js`에 지정되어 있습니다 (`enemy_01.png` ~ `enemy_06.png`)
3. 파일명만 맞추면 자동으로 전투 화면에 카드형으로 표시됩니다

이미지가 없으면 이모지로 자동 대체됩니다.

---

## 🔊 사운드 파일 추가 방법

`sounds/` 폴더에 mp3 파일을 파일명에 맞게 넣으면 자동 연결됩니다.
파일이 없어도 게임은 정상 작동합니다 (무음 상태).

## 💾 저장 시스템

- `localStorage`에 자동 저장 (키: `seedGame_v3`)
- 저장 항목: 플레이어 이름, 스테이지 진행 상황, 정답/오답 수
- 타이틀에서 **이어하기** 버튼으로 이어서 플레이 가능

---

## 📝 오답노트

- 전투에서 틀린 문제는 자동으로 오답노트에 저장됩니다
- 타이틀 메뉴 **오답노트** 버튼에서 언제든 확인 가능합니다
- 최대 50개까지 저장되며 초기화 가능합니다

---

## 🚀 GitHub Pages 배포 방법

1. 이 폴더 전체를 GitHub 저장소에 업로드
2. Settings → Pages → Branch: main, folder: / (root) 선택
3. 저장 후 생성된 링크 공유

> `images/`와 `sounds/` 폴더도 반드시 함께 올려야 합니다.
